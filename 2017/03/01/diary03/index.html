Day18： 20170302

1、完成了SQL和统计基础部分的学习，了解了查看统计图的方法。确定统计图可以通过位置和离散程度两大因素确定，其中曲线位置可以根据均值、中位数、众数来确定，离散程度可以根据峰度、偏度、百分位数来确定。
2、最近，在对新媒体的研究过程中学习到了推荐算法，推荐算法大致分为三类：基于内容的推荐算法、协同过滤推荐算法和基于知识的推荐算法。
基于内容的推荐算法主要用于资讯类，原理是用户喜欢和关注的Item在内容上与之有关联性的推荐给用户，可以避免冷启动的问题，弊端是可能会重复推荐。
协同过滤推荐算法主要用于电商类，原理是把具有相似兴趣用户喜欢过的商品推荐给用户；还有一种是基于Item的协同过滤算法。这两种方法都是将用户的所有数据读入到内存中进行运算。
基于知识的推荐算法主要用于广告行业，原理是构建领域本体或者建立一定的规则，进行推荐。

Day19： 20170305

1、周末学习了R中的分组统计语句和统计学中的参数估计与假设检验。在R中，如果是非常复杂的数据处理，可以用多个语句执行，非常方便。此次学习的统计内容，主要分为描述性统计和统计预测两大分支，因为私事落了一天课程，这个礼拜任务很重，要坚持弄明白。
2、正好项目上需要写个统计关键词的脚本，就拿python来实现练练手。周五实现了python连接mysql数据库的功能，使用到的类库是MySQLdb。具体实现代码如下：

import MySQLdb as mdb

try:
    conn = mdb.connect(
            host='127.0.0.1',
            port = 3306,
            user='user01',
            passwd='password01',
            db ='db01',
            charset='utf8'
            )
except:
    print "Could not connect to MySQL server."
    exit(0)

try:
    cursor = conn.cursor()
    TABLE_NAME = 'zk_article'
    count = cursor.execute('SELECT keywords FROM %s WHERE is_open = 1' %TABLE_NAME) #执行结果count是返回记录数
    results = cursor.fetchall() #获取返回数据集
    results = list(results) #数据集处理
    for row in results:
        kw_str = ('%s' % row)
        print kw_str
except:
    import traceback
    traceback.print_exc()
finally:
    cursor.close()
    conn.close()
