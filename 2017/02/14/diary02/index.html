成功日记2月

发表于 2017-02-14   |   分类于 成功日记

Day7： 20170213

1、今天是农历年放假后上班的第一天，这个长假好好的休息了一把，要快速收心开始着手为自己的目标而努力了。今年的目标非常明确–英语和文章，都是上半年需要完成的目标，工作的事情特别多，所以留给自己的时间有限。每天努力一点点，相信自己能够达成目标和理想！
2、今天下载了写关于CNN的论文，感觉这块大有写头以及研究的价值，而Python语言也能应用其中，可谓一举两得。目前重新调整了文章的方向和思路，仍然在学习和资料收集的阶段，这个过程漫长而枯燥，希望自己能够不忘初心继续前进，能够取得成果。
3、想要学习一门语言和学科，还是得从实践出发，理论看了一些仍然云里雾里。目前，选中了自然语言处理中的情感分析方向为切入口，对CNN和Python进行深入学习和技能实践。根据论文描述，学习过程为：京东商品用户评论数据集（包括正负样本）-10折交叉验证划分数据集-数据预处理（过滤标点符号和特殊字符，格式转换，分词处理，训练word embedding）-模型训练。今天确定了学习思路和方法，明日开始着手第一步的实践，即使用Python语言抓取京东商品用户评论，依次进行，逐步实现。

Day8： 20170214

1、今天主要在研究python对网页数据的采集–以京东商品评论为例，并实现了简易数据的采集。
2、在学习中了解到，除了python自带的模块外还需要使用第三方模块，而且python有丰富的第三方模块，可以极高的提升开发效率。下载第三方模块就需要使用到python包管理工具，python自带的管理工具为easy_install，目前常用的为pip，可完全取代easy_install。第三方python包的安装方法非常简单，在pip的目录下打开cmd,输入命令“pip install packagename”即可。
3、在采集方面，beautifulSoup4和requests是应用于采集的python包，二者使用easy_install或者pip安装即可。requests是python的一个HTTP客户端库，与urllib、urllib2类似，代码却简单的多。beautifulSoup是用于解析HTML/XML的python包，可以很好的处理不规范标记并生成剖析树，它提供简单又常用的导航、搜索以及修改剖析树的操作，大大节省编程时间。
4、今天解决了使用mysql语句获取间隔时间的问题，即库中记录的时间戳与当前时间的间隔。timediff函数用于获取间隔时间，格式为111：00：00(即时：分：秒)；datediff函数用户获取间隔天数。具体实现代码为：

timediff(now(),FROM_UNIXTIME(add_time,'%Y-%m-%d %H:%i')) AS gapHour

Day9： 20170220

1、近日，主要实现媒体平台中第三方人员的文章管理功能。即第三方人员可以查看所有文章，但只能管理自己发布的文章内容；内部管理人员可以管理第三方人员发布的文章，并进行审核通过，只有审核通过的文章才能在前端发布。而这角色的控制就是权限管理，ecshop的权限管理功能还是很好的，直接在表中读取各个用户的权限，而代码根据权限来执行相对应的功能。
2、本周正式开始了《自然语言处理》这本书的学习，通过书本来系统的掌握NLP的整个流程和方法，为后续的实践处理打下基础。目前还处在第三章即数据的处理阅读中，讲的很基础，结合了Python语言的基本语法。打算花费一个月时间来学习该本书，并进行实践操作。
3、又重新开始了Keep的训练，每天运动一会会，肩膀也没有之前疼了，刚上班的那天疼的受不了，没想到每天只要花费这么短的时间就能改善缓解疼痛，一定要坚持下去！

Day10： 20170221

1、今天使用Python实现了对CSV文件内容的读取，而完成此项功能需要用到pandas包。pandas是基于Numpy构建的含有更高级数据结构和工具的数据分析包，类似于Numpy的核心是ndarray，pandas也是围绕着Series和DataFrame两个核心数据结构展开的。Series和DataFrame分别对应于一维的序列和二维的序列。
pandas约定俗成的导入方式为：

from pandas import Series,DataFrame
import pandas as pd

读取CSV文件（其他文件类型类似）:

content = pd.read_csv(r'file.csv',encoding="gbk")

获取指定字段：

comments = content['content']

2、今天晚上用半个小时学习了NLP第三章数据处理其中的正则表达式，即如何用正则过滤字符串并进行分词，前几章内容只是基本的技术介绍。
3、花费了半个小时阅读英文文章两页，虽然还是有许多不认识的单词和表达方式，但是在理解上增强了一丢丢。在阅读的过程中发现，可能本书的主人翁对话的对象是孩子，所以对话内容都能看懂，而叙述部分有许多生僻单词。
