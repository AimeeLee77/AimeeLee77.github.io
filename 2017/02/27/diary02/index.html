1、使用pandas和stirng包实现字符串的处理。

line = line.rstrip() #去除文本后的'\n'
intab = ""
outtab = ""
trantab = string.maketrans(intab, outtab)
pun_num = string.punctuation + string.digits
line = line.encode('utf-8')
line = line.translate(trantab,pun_num)

Python maketrans() 方法用于创建字符映射的转换表，对于接受两个参数的最简单的调用方式，第一个参数是字符串，表示需要转换的字符，第二个参数也是字符串表示转换的目标。
注：两个字符串的长度必须相同，为一一对应的关系。然后可以使用translate()方法来使用。

String模块中的常量：
string.digits：数字0~9
string.letters：所有字母（大小写）
string.lowercase：所有小写字母
string.printable：可打印字符的字符串
string.punctuation：所有标点
string.uppercase：所有大写字母

去除中英文标点符号如下：
line = line.decode(‘utf-8’)

line = re.sub("[\s+\.\!\/_,$%^*(+\"\']+|[+——！，。？?、~@#￥%……&*（）]+".decode("utf8"), "".decode("utf8"),line) #去除文本中的中文符号和英文符号

Day12： 20170223

1、对于中文文本的数据预处理，在完成标点符号、特殊字符的处理后，就需要进行分词操作。使用了目前对中文处理效果较好的jieba(结巴)分词。
通过查阅资料了解到，jieba分词支持三种分词模式：

  • 精确模式。试图将句子最精确的切开，适合文本分析；
  • 全模式。把句子中所有可以成词的词语都扫描出来，速度非常快，但是不能解决歧义；
  • 搜索引擎模式。在精确模式的基础上，对长词再次切分，提高召回率，适用于搜索引擎分词。
    主要采用一下算法：
  • 基于Trie树结构实现高效的词图扫描，生成句子中汉字所有可能成词情况所构成的有向无环图（DAG）
  • 采用动态规划查找最大概率路径，找出基于词频的最大切分组合

  • 对于未登陆词，采用基于汉字成词能力的HMM模型，使用Viterbi算法

    import jieba

seg_list = jieba.cut(line,cut_all=False)
print "Default Mode:","/".join(seg_list)
