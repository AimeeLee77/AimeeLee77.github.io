Day156: 20171024

1、今天把云图实例又整理了一遍，基本熟悉了整个绘制的流程，现把绘制词云的步骤和关键知识点总结如下。

  • 1）采集数据。

在本例中我采集的是豆瓣《王牌特工2：黄金圈》这部电影的部分评论数据，因为网站限制采集了大约200条评论作为数据集。核心提取字段代码为：

movie_items = re.findall(r'<div.*?class="comment".*?<p.*?class.*?>(.*?)</p>.*?</div>', my_page, re.S)
  • 2）数据处理。

将数据集做以下处理：合并所有评论数据；去除停用词（符号、停用词表、自定义停用词词库）；分词。其中，分词采用结巴分词。本例中为了方便舍弃了去除停用词的步骤，在实际的应用中还是需要加入的，而且构建自定义停用词还是蛮重要的，jieba程序包中提供了相应的接口进行实现。

  • 3）提取关键词。

提取关键词最常采用的是TF-IDF（词频-逆向文件频率）算法策略，jieba软件包中也提供了此项功能，通过代码

jieba.analyse.extract_tags()

即可实现。本例未进行此步骤的操作，只是拿来了全部的分词结构进行词云构建，因此具体实现方法待以后做实际项目中再进行深入学习。

  • 4）绘制词云。

得到关键词列表后即可进行词云的绘制，需先下载第三方库wordcloud，具体步骤是：可先设置背景图片，图片深色部分即为绘制区域，不设置的话默认是矩阵形式；第二步，构建词云图，通过WordCloud().generate(word_split)函数实现，其中参数word_split为关键词列表，通过空格隔开；最后，绘制图形，通过matplotlib.pyplot工具包实现，猜测可能有更漂亮的实现形式，待以后发现，同时，wordcloud还提供了根据背景图片颜色创建相似颜色的功能，采用ImageColorGenerator()函数即可；若需要保存生成的词云，可使用word_cloud.to_file()函数实现。

# 设置背景图片
color_mask = imread("xin.jpg")
# 获取停用词
stopwords = [w.strip() for w in open("stopWord.txt","r").readlines()]
#print stopwords
#sys.exit()

word_cloud = WordCloud(font_path='simsun.ttc',
                       background_color="white", max_words=1000, mask=color_mask,
                       stopwords=stopwords, max_font_size=40, random_state=42)  # wordcloud参数定义
word_cloud.generate(word_split)  # 生成词云图

# 画图
plt.imshow(word_cloud, interpolation="bilinear")
plt.axis("off")
plt.figure() # 绘制图形
# recolor wordclound and show
image_colors = ImageColorGenerator(color_mask) # 根据图片创建颜色
plt.imshow(word_cloud.recolor(color_func=image_colors),interpolation="bilinear")
plt.axis("off")
plt.figure()
