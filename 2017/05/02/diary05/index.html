Day49: 20170502

1、今天看了一篇介绍十大机器学习算法的文章，对基本的机器学习算法概念有了一定的了解，后续就是各个击破了解原理与python实现方法。现将这些算法做个回顾：

  • 决策树。最基本的机器学习算法，对特征进行分类，每一个树节点提一个问题，根据问题将数据分成两类，逐步形成一颗茁壮的二叉树。
  • 随机森林。决策树算法的衍生，从原始数据集中随机选取M组数据，得到M个决策树，将新的数据分别放入这M棵决策树中得到M个分类结果，选取类别最多的哪一类作为最终类别即取众数。
  • 逻辑回归。在线性分类模型的基础上满足两个条件：大于等于0，小于等于1，根据数学计算变形可得到逻辑回归模型。
  • SVM。将N类分开需要得到一个超平面，最优超平面是使得N类的margin达到最大，margin是超平面与各类离他最近的一个点的距离，即变成了求最优解的问题。
  • 朴素贝叶斯。即各个类中出现的概率，选择概率值最高的那一类为最终分类结果。
  • K最近邻。距离新数据最近的K个点中，哪个类别的个数最多就属于那一类。
  • K均值。先随机选取K个分类的初始值，剩下的每个数据都与K个初始值计算距离，归纳到距离它最近的那个初始值所属的类别，分好类后计算每个类别的均值，并最为新一轮的中心点，几轮后分组不再变化即可停止。
  • Adaboost。把若干个效果并不好的分类器组合起来考虑，会得到一个效果比较好的分类器。
  • 神经网络。包括输入层、隐藏层和输出层，实际是误差的反向传播。
  • 马尔可夫。先给每个单词设定一个状态，计算状态间转换的概率。

  • 机器学习是人工智能的第一个浪潮，机器学习属于浅层学习，只含有一层隐层或者没有隐层；深度学习是人工智能的又一浪潮，深度学习属于深沉学习，含有多个隐层（5、6、10甚至更多）。
  • 机器学习属于有监督学习，通过误差的反向传播来调整参数；深度学习是无监督学习，本质是有层级关系的抽象和迭代过程，即每一层的输出都是下一层的输入，且同层和跨层之间的节点没有联系。
  • 算法的准确性取决于特征的表达，大部分的工作也在于此。
