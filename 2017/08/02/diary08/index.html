成功日记2017年8月

发表于 2017-08-02   |   分类于 成功日记

Day109: 20170801

1、今天开始阅读导论第九章的内容，依然是懵懵懂懂，第二节主要介绍了基于原型的几种聚类算法：模糊c均值、使用EM算法的混合模型聚类，以及自组织映射（SOM）。还是那句话，第一遍先了解主要内容以及脉络结构，待以后再细细品读理解并运用。
2、转眼就到了8月，不知是不是因为给自己定制了目标，觉得时间过得尤其快，6、7两个月的目标，导论即将在本周完成勉强算完成目标，而论文只执行了一部分内容，算未完成，休假回来已是8月中旬，计划在十一前完成论文的所有工作，专注于此，心无旁骛！

Day110: 20170802

1、今天粗略的阅读完了第九章聚类算法的内容，主要介绍了基于原型、基于密度、基于图和可伸缩的几种聚类算法，这些高级算法目前来说还不能看懂，待以后从实践中学习理解。明天完成最后一章异常检测的内容，该本书就算正式结束了，但是时间太长前面内容都忘记差不多了，有种从未读过的感觉，之后的学习就是缺哪补哪，这是本值得多次精读的书籍！
2、转眼已经来到下半年，我也将正式启动自己的英语学习计划，PETS5下半年报名时间在9-10月份，考试时间为12月份，这次一定不能错过和放弃了。先计划每天来办公室的前半个小时背诵单词和句子，找回语感并扩大英语词汇量，后续再另行安排，万事开头难，一定要坚持下去。

Day111: 20170803

1、今天结束了导论第一遍的浏览，知道了整本书的脉络结构，分为六大部分：数据挖掘概念、数据、分类算法、关联分析、聚类算法和异常检测，同时了解了各个模块的常用算法。接下来的工作就是针对每个重要部分进行精读，并做好相对应的笔记，多看多用多理解，看了其它基本推荐的书，如果这本书吃透下来了，数据挖掘的内容就清楚了绝大部分。
2、明天整理下论文方面的资料和内容，下周休假，休假归来后就开始着手论文和英语学习两项任务。
