成功日记2017年7月

发表于 2017-07-03   |   分类于 成功日记

Day90: 20170702

1、周六花了好长时间来研究CNN的算法，找到了其他的简单CNN来处理文本分类问题，但是还没有嫁接到自己的数据。这周在上班时间继续研读CNN理论的文章并进行写作，下班则重点完成代码部分，还是需要花一定的时间和精力在上面才能够完成任务。

Day91: 20170703

1、今天把服务器的PHP版本升级到了PHP7，又多了修改原始网站代码的事情。目前来说，主要有3件事情，第一件也是目前最重要的是代码修改，尤其是数据库读取的代码；第二件是政策库数据清洗；第三件是将政策库清洗后的数据移植到华为云服务器上。
2、今天看了几章数学之美的内容，整体读下来感觉作者讲的内容很多，每个模型或者方法都值得深入的去了解和研究，尽快把后面的内容结束掉，并做好总结。从这一段时间的安排来看，晚上如果运动了就没有办法静下心来学习，二者只能选择其一，只能将晚上学习时间移动到白天，待数学之美这本书看完后，就开始着手导论的持续性学习了，坚持。
3、晚上去民大的操场跑了3.5公里，虽然很累但是真的很爽，自己的身体也能够支持这种强度的训练，以后还是需要跑步和跳操相结合，做到每天都出汗。

Day92: 20170704

1、今天把服务器上的代码改了，重点是PHP7把mysql废弃改成了mysqli，其他的重点就是solr、memcache等第三方库的部分，这个待后续慢慢修改。
2、今天读了数学之美和导论的部分内容，上班时间读书的效率真的很高，很容易静下心来，之后继续此种方式，同时还需要开始论文部分了。

Day93: 20170705

1、今天阅读到了吴军博士另一本书《大学之路》中的一段话，感触颇深，现摘录如下。

在吴军的印象里，父母晚上从不应酬，也不看什么电视，总是非常有规律地学习。吴军兄弟认为，父母的身教潜移默化地影响了他们，让他们养成了终身学习的习惯。而这一点，至关重要。在吴军看来，人生是一场马拉松，起跑的那一瞬间道路是非常拥挤的，但跑完1/4以后，选手们的距离就拉开了，起跑时占得的一点点先机此时已荡然无存了。“很多中国的家长都在说不能让孩子输在起跑线上，因此他们会让孩子在起跑线上尽可能地抢位子。但其实，成功的道路并不像想象得那么拥挤，因为在人生的马拉松长路上，绝大部分人跑不到一半就主动退下来了。到后来，剩下的少数人不是嫌竞争对手太多，而是发愁怎样找一个同伴陪自己一同跑下去。因此，教育是一辈子的事情，笑到最后的人是一辈子接受教育的人。”

2、今天修改了论文实验的代码部分，还是取得了一定的进展的，明天继续。在下班前只留下了10几分钟时间，却也看了导论中一点点的内容，还是这样学习效果比较好，之后每天多留一点时间至少1个小时给导论的阅读，相信这个月能够完成原定的计划，加油。

Day94: 20170706

1、今天把政策库数据清洗的代码已经写好并测试完成了，同时标题符号清洗已经完成，目前就只剩下正文填充，而正文填充需要读取168服务器的mongodb，该服务器目前无法访问系统部的也没有回应，只能等明天服务器可以访问后进行代码运行和数据的备份与迁移了。
2、今天正式结束了数学之美这本书的第一遍，算是浏览的一个过程，大致知道了整本书的内容和写作风格，接下来就是第二遍精读的过程，并把整本书的框架做成思维导图，以便之后查阅。这本书讲了很多的数学模型和实现方法，并且很多数学方法都是触类旁通的，复读和总结的过程是消化并转成自己知识的必备过程，预计这周末完成。
3、这两天读了导论第五章最近邻分类器和贝叶斯分类器两个部分的内容，说实话并没有掌握，专业书籍的概念和公式太多了，让我有点犯迷糊，但是我决定不能因为不懂而停止，继续将这一章后面部分的内容结束掉，再回过头来将整个分类重新阅读并理解一遍，后面的关联和聚类也用该种方式进行。该本书重点就是分类、关联和聚类，还有两小点是数据和异常点检测，把前三个重点研究透彻后即把该本书吃透了，加油。

Day97: 20170710

1、今天上午自习研读了一篇关于CNN的综述论文，收获很多，有许多值得借鉴的地方，尤其是模型各个组成部分的数学描述。该论文主要是讲CNN的详细内容，明天继续阅读其他论文，了解CNN相关研究部分的写作方法。
2、下午大约花费了一个小时阅读导论第五章的支持向量机算法部分，文章首先讲述了SVM线性可分情况下的数学推导；然而真实场景中往往是线性不可分的，解决方法是讲输入数据x通过非线性变换，从原始特征空间映射到高维空间，使得决策边界在高维空间中是线性的；而映射到高维空间，若维度过高将产生“维度灾难”的问题，因此引入核技术，因为核函数是在原始属性空间中计算相似度的函数，因此避免了“维度灾难”问题的发生。

1、上午阅读了CNN相关的论文，大部分都是在阐明数学模型，对我的第三章写作内容帮助很大，明天继续阅读CNN原理部分，争取这周把第二章的写作任务完成。
2、今天阅读完了分类技术中的组合方法，主要介绍了通过处理训练数据集的两种组合方法-装袋和提升，以及通过处理输入特征的组合方法-随机森林。该章内容只剩下最后一点知识点，明天完成该章后即开始关联分析部分的学习了，虽然之前的内容都似懂非懂，甚至有些已经忘记了，但是进度不能停，待完成全部内容的第一遍学习、浏览后，再进行第二遍的精读。

Day99: 20170712

1、今天结束了导论中的分类部分，并开始了关联分析部分，阅读了第六章的前两小节。对于关联分析，之前知之甚少，却发现自己静下心来也能够读懂默生的专业术语、原理、方法，虽然很容易忘记，却是了解它的第一步，继续努力，每天多看一点点，随着知识的累积和理解的深入，相信总可以成为合格的分析人员。

2、转眼已经是写日记100的日子了，虽然耗时了半年，但是我还是坚持下来了，很为自己骄傲的。从开始的零星日记，到后来断断续续的记录，目前真的是已经养成了每天上班第一件事情就打开文档写日记的习惯了，每天不写点总觉得忘记了什么事情，浑身极不舒服。习惯的培养真的需要坚持和毅力，我用实际行动证明了自己是可以做到的，很喜欢现在自己的模式，每天都很充实，每天都能学到新的东西、有所收获，继续努力，未来还有更多好的习惯让我去坚持，勿忘初心不负梦想！

Day101: 20170717

1、忙忙碌碌的周末和周一，基本没有搞学习的事情，不过岗聘的材料和PPT都已经完成了，这几天好好准备下报告的语言组织和大数据相关的一些理论知识，为岗聘汇报做好准备。另外，今天抽空看了下导论中关联分析的内容，发现越来越看不懂了，计划将第六章读完后先阅读聚类部分的内容，再回过头来完成第七章的阅读。而且，由于各种事情的堆置和处理，以及自己的懒惰，论文和导论两项任务都在7月完成的可能性为零，因此决定全力以赴先实现导论阅读这项目标，如果肯花时间一定可以做到，待8月份开始专攻论文，看来我还是得一件件的做事，不能一起做，要不然就会很难完成，导论加油。

Day102: 20170718

1、今天结束了导论第六章内容的阅读，并完善了该部分的脑图。第六章主要讲述了关联分析的基本概念和专业术语，并指出关联分析算法主要是解决频繁项集的产生和规则的产生两个子问题，同时介绍了频繁项集产生的算法、规则产生的算法以及频繁项集的紧凑表示，介绍了FP算法和FP树，最后介绍了关联模式的评估方法以及倾斜支持度分布的影响。明天继续阅读关联分析的高级部分，争取早日完成。脑图的存在真的非常有价值，本来已经看这一章看晕了，完全理解不了，经过绘制脑图，整个章节的写作框架和知识体系就出来了，再看每一个分支的内容也就轻松许多，继续努力，每天进去一点点。

Day103: 20170719

1、今天开始导论第七章关联分析高级概念的阅读，本章主要讲解了对称二元属性、分类属性和连续属性数据集的关联分析方法。7月只剩下最后的10天时间，本周还剩两天，计划本周完成第七章的阅读。
2、工作内容方面，需完成以下几项：政策列表需提取1000条数据放入新建表中，用于首页访问；政策正文表需横向切表，每张表存放100W条记录；政策规划库根据表结构的调整修改读取代码；研究可视化引擎技术和解决方案。

Day104: 20170720

1、今天阅读完成了导论第七章的内容，至此关联分析部分全部阅读完成，纵观整个内容主要讲解了两种关联规则分析方法：Apriori算法和FP增长算法。因为之前对关联分析并不了解，因此即使看完了这一部分的内容依然觉得云里雾里，因此只是大致浏览了下该部分，知道了关联分析主要是干什么的、怎么做的，待以后再来进行精读与实践的结合。明天继续第八章聚类分析的内容，努力。
