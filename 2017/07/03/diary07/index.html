成功日记2017年7月

发表于 2017-07-03   |   分类于 成功日记

Day90: 20170702

1、周六花了好长时间来研究CNN的算法，找到了其他的简单CNN来处理文本分类问题，但是还没有嫁接到自己的数据。这周在上班时间继续研读CNN理论的文章并进行写作，下班则重点完成代码部分，还是需要花一定的时间和精力在上面才能够完成任务。

Day91: 20170703

1、今天把服务器的PHP版本升级到了PHP7，又多了修改原始网站代码的事情。目前来说，主要有3件事情，第一件也是目前最重要的是代码修改，尤其是数据库读取的代码；第二件是政策库数据清洗；第三件是将政策库清洗后的数据移植到华为云服务器上。
2、今天看了几章数学之美的内容，整体读下来感觉作者讲的内容很多，每个模型或者方法都值得深入的去了解和研究，尽快把后面的内容结束掉，并做好总结。从这一段时间的安排来看，晚上如果运动了就没有办法静下心来学习，二者只能选择其一，只能将晚上学习时间移动到白天，待数学之美这本书看完后，就开始着手导论的持续性学习了，坚持。
3、晚上去民大的操场跑了3.5公里，虽然很累但是真的很爽，自己的身体也能够支持这种强度的训练，以后还是需要跑步和跳操相结合，做到每天都出汗。

Day92: 20170704

1、今天把服务器上的代码改了，重点是PHP7把mysql废弃改成了mysqli，其他的重点就是solr、memcache等第三方库的部分，这个待后续慢慢修改。
2、今天读了数学之美和导论的部分内容，上班时间读书的效率真的很高，很容易静下心来，之后继续此种方式，同时还需要开始论文部分了。

Day93: 20170705

1、今天阅读到了吴军博士另一本书《大学之路》中的一段话，感触颇深，现摘录如下。

在吴军的印象里，父母晚上从不应酬，也不看什么电视，总是非常有规律地学习。吴军兄弟认为，父母的身教潜移默化地影响了他们，让他们养成了终身学习的习惯。而这一点，至关重要。在吴军看来，人生是一场马拉松，起跑的那一瞬间道路是非常拥挤的，但跑完1/4以后，选手们的距离就拉开了，起跑时占得的一点点先机此时已荡然无存了。“很多中国的家长都在说不能让孩子输在起跑线上，因此他们会让孩子在起跑线上尽可能地抢位子。但其实，成功的道路并不像想象得那么拥挤，因为在人生的马拉松长路上，绝大部分人跑不到一半就主动退下来了。到后来，剩下的少数人不是嫌竞争对手太多，而是发愁怎样找一个同伴陪自己一同跑下去。因此，教育是一辈子的事情，笑到最后的人是一辈子接受教育的人。”

2、今天修改了论文实验的代码部分，还是取得了一定的进展的，明天继续。在下班前只留下了10几分钟时间，却也看了导论中一点点的内容，还是这样学习效果比较好，之后每天多留一点时间至少1个小时给导论的阅读，相信这个月能够完成原定的计划，加油。

Day94: 20170706

1、今天把政策库数据清洗的代码已经写好并测试完成了，同时标题符号清洗已经完成，目前就只剩下正文填充，而正文填充需要读取168服务器的mongodb，该服务器目前无法访问系统部的也没有回应，只能等明天服务器可以访问后进行代码运行和数据的备份与迁移了。
2、今天正式结束了数学之美这本书的第一遍，算是浏览的一个过程，大致知道了整本书的内容和写作风格，接下来就是第二遍精读的过程，并把整本书的框架做成思维导图，以便之后查阅。这本书讲了很多的数学模型和实现方法，并且很多数学方法都是触类旁通的，复读和总结的过程是消化并转成自己知识的必备过程，预计这周末完成。
3、这两天读了导论第五章最近邻分类器和贝叶斯分类器两个部分的内容，说实话并没有掌握，专业书籍的概念和公式太多了，让我有点犯迷糊，但是我决定不能因为不懂而停止，继续将这一章后面部分的内容结束掉，再回过头来将整个分类重新阅读并理解一遍，后面的关联和聚类也用该种方式进行。该本书重点就是分类、关联和聚类，还有两小点是数据和异常点检测，把前三个重点研究透彻后即把该本书吃透了，加油。

Day95: 20170707

1、今天把工作上手头的事情基本都结束掉了，包括169服务器代码的修改，最重要的就是mysql和mongodb数据库的连接、读取；solr服务器和mongodb数据库的访问；政策规划库数据的清洗全部完成并且已经在服务器上备份好。工作上的事情算是有了一个比较好的了解，下周还是以论文为主，看书为辅。
